# Student Math Performance Prediction 🎓📊

This repository contains a Machine Learning project focused on predicting student academic performance in mathematics based on various demographic, social, and school-related features.

---

## 📌 Project Overview
Predicting student performance helps educators identify students who might need extra support early in the academic year. This project utilizes Python and popular machine learning libraries to analyze student data, perform exploratory data analysis (EDA), and build predictive models.

## 🛠️ Tech Stack & Libraries
* **Language:** Python
* **Data Analysis & Manipulation:** NumPy, Pandas
* **Data Visualization:** Matplotlib, Seaborn
* **Machine Learning:** Scikit-learn

## 📂 Project Structure
* `Student_Math_Prediction_Group...` - Jupyter Notebook containing data preprocessing, EDA, model training, and evaluation.
* `README.md` - Project documentation.

## 🚀 Key Features
1. **Data Preprocessing:** Handling missing values, encoding categorical variables, and feature scaling.
2. **Exploratory Data Analysis (EDA):** Visualizing correlations between student backgrounds (e.g., study time, parental education) and their final grades.
3. **Model Training:** Training multiple ML algorithms to find the most accurate predictive model.
4. **Evaluation:** Assessing models using metrics like Mean Absolute Error (MAE) and $R^2$ Score.
